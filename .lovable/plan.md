

# Revuest — Reputation Management Website

A single-page marketing website for Revuest, a UK-based review automation agency. Clean, professional, conversion-focused design using Open Sans font, primary blue (#0F57EA), and light blue secondary (#E9F0FF).

## Pages & Sections

### 1. Hero Section
- Bold headline "Review Automation for Your Business" with subheading
- Two CTA buttons: "Start Getting Reviews" (filled blue) and "Book a Demo" (outlined)
- Clean, spacious layout with subtle background pattern or gradient using the secondary color

### 2. Problem Section
- Headline with relatable narrative about happy customers not leaving reviews
- Three problem cards in a grid layout with icons, highlighting pain points (not enough reviews, no time, low visibility)

### 3. Solution Section
- "How Revuest Fixes That" with four benefit cards in a grid
- Each card with an icon and short description (Win Customers, Climb Rankings, Build Trust, Stand Out)

### 4. Results Timeline
- "What Can Happen In The First 7 Days" with CTA buttons
- Visual timeline showing Day 1, Day 3, and Day 7 milestones with a connecting line/step indicator

### 5. How It Works
- Three-step process: Connect → Launch → Automate
- Numbered cards with clean iconography

### 6. Features Grid
- Six feature cards in a responsive grid (AI Responses, Smart Follow-Ups, Review Reactivation, Image Requests, Social Posting, Website Widgets)

### 7. Pricing Section
- Single "Growth Plan" pricing card (£179/month)
- Feature checklist with checkmark icons
- "Start Free Trial" CTA button

### 8. FAQ Section
- Accordion-style FAQ with 7 questions covering target audience, integrations, setup, timeline, contracts, policies, and UK support

### 9. Navigation & Footer
- Sticky top navigation with Revuest logo, section links, and CTA button
- Footer with company info, links, and copyright

## Design System
- **Font**: Open Sans with varying weights (700 for H1, 600 for H2, etc.)
- **Primary**: #0F57EA (buttons, accents, headings)
- **Secondary**: #E9F0FF (card backgrounds, section alternation)
- **Cards**: Subtle shadows, rounded corners, hover effects
- **Responsive**: Mobile-first, stacking grids on smaller screens
- Smooth scroll navigation between sections
